# Tobi Landing Page

Public landing page for [Tobi](https://tobi.fun) - Your AI-powered web3 assistant.

## About

Tobi is the world's first AI-powered web3 assistant, built right into Telegram, making web3 as easy as chatting with your friends.

## Tech Stack

- **Framework:** Vue 3
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Hosting:** GitHub Pages

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
